A comprehensive comparison platform for exploring the capabilities of modern text-to-image diffusion models. Generate, compare, and analyze outputs from 14 state-of-the-art models through an intuitive web interface.

About This Project

This repository provides a comprehensive case study of existing open-source diffusion models' capabilities in text-to-image generation. The project features an intuitive Gradio web interface that allows users to generate images with multiple models simultaneously, compare results side-by-side, and explore the unique strengths of each architecture.

Key Features:

  • 14 State-of-the-Art Models: Including FLUX.1 Dev, Stable Diffusion 3, SDXL, CogView4, HunyuanDiT, PixArt-Sigma, and more
  • Interactive Web UI: Built with Gradio for easy experimentation and comparison
  • Configurable Parameters: Control inference steps, guidance scale, resolution, and seed for reproducible results
  • Gallery View: Compare outputs from multiple models with labeled results
  • Auto-Save Outputs: All generated images are automatically organized by model with metadata
  • Memory Efficient: Sequential generation with intelligent model caching

Whether you're a researcher comparing model architectures, an artist exploring AI-generated imagery, or a developer building image generation applications, this tool provides a unified platform to experiment with the latest diffusion models.

Supported Models

Stable Diffusion Series: Classic SD 2.1, high-quality SDXL, and cutting-edge SD3 Medium for reliable text-to-image generation.

State-of-the-Art Models: FLUX.1 Dev for exceptional quality, HunyuanDiT for multilingual support, and Stable Cascade for multi-stage generation.

Efficient Architectures: PixArt-XL, PixArt-Sigma for fast generation, Sana 600M for lightweight deployment, and Lumina-Next SFT.

Specialized Models: CogView3 Plus and CogView4 6B for multilingual capabilities, Qwen Image, and UniDiffuser v1 for unified multimodal generation.

Getting Started

Installation

# Clone the repository
git clone https://github.com/Bili-Sakura/image-generation-case-study.git
cd image-generation-case-study

# Install dependencies
pip install -r requirements.txt

Quick Start

Option 1: Gradio Web UI (Recommended)

# Launch the interactive web interface
python run.py

# Or run directly
python -m src.app

This will open a web browser at http://localhost:7860 with an intuitive UI for generating and comparing images.

Option 2: Developer Mode

# Launch with manual model selection
python run.py --dev

Allows you to select specific models before loading (runs on port 7861).

Option 3: Python API

from src.model_manager import get_model_manager
from src.inference import generate_image

# Load model
manager = get_model_manager()
manager.load_model("stabilityai/stable-diffusion-2-1")

# Generate image
image, filepath, seed = generate_image(
    model_id="stabilityai/stable-diffusion-2-1",
    prompt="A fantasy landscape with mountains and rivers",
    num_inference_steps=50,
    guidance_scale=7.5,
    seed=42
)
